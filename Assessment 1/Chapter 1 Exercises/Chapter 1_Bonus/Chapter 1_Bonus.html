HTML Tags

Structure & Text

<h1> to <h6> - Headings

Defines the titles and subtitles of sections on a webpage. There are six levels of headings in HTML, with <h1> being the highest (or most important) and <h6> the lowest.

<h1>Main Heading</h1> 
<h2>Sub Heading</h2>

<p> - Paragraph

Defines a paragraph of text which helps to divide content into manageable and readable sections.

<p>This is a paragraph of text.</p>

<div> - Division

A block-level container used to group content together and can be used to apply styles or manipulate content with CSS or JavaScipt.

<div>
  <p>Content inside a div.</p>
</div>

Links & Images

<a> - Anchor (Link)

Creates a hyperlink that can be used to link to a specific location within the same page or to another page entirely.

<a href="https://www.google.com">Visit Google</a>

<img> - Image

Embeds an image into the page. Requires 'src' (source) and 'alt' (alternative text) attributes.

<img src="image.jpg" alt="Description of image">

Lists

<ul> - Unordered List

A collection of items displayed with bullet points.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol> - Ordered List

Displays items in an ordered form, either numerical or alphabetical.

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

<li> - List Item

Defines an item within an ordered (<ol>) or unordered (<ul>) list.

<li>This is a list item.</li>
